# 天堂W打王重生計時器 - 設計風格構思

## 三種風格方向

### 方向一：暗黑地城風 (Dark Dungeon)
**簡介：** 以深色為主調，搭配火焰橙紅色點綴，營造出地城深處的緊張感與戰鬥氛圍。彷彿玩家正在地城中等待 Boss 重生。
**機率：** 0.07

### 方向二：戰術指揮台 (Tactical Command)
**簡介：** 模擬軍事指揮中心的 HUD 介面，以深藍黑底搭配青綠色數據光效，呈現出精密的戰術規劃感。
**機率：** 0.05

### 方向三：中世紀卷軸風 (Medieval Scroll)
**簡介：** 以羊皮紙質感為底，搭配手繪風格的裝飾邊框與哥德字體，營造出奇幻冒險世界的古典氛圍。
**機率：** 0.03

---

## 選定方向：暗黑地城風 (Dark Dungeon)

### Design Movement
Dark Fantasy UI — 結合遊戲 HUD 設計與暗黑奇幻美學，參考暗黑破壞神系列與天堂系列的介面風格。

### Core Principles
1. **沉浸感優先** — 整體視覺必須讓玩家感覺身處遊戲世界中
2. **資訊密度高但清晰** — Boss 資訊一目了然，重要數據突出顯示
3. **動態回饋** — 計時器倒數、狀態變化都有視覺動畫回饋
4. **層次分明** — 透過光影與邊框區分不同資訊區塊

### Color Philosophy
- **主背景：** 深灰黑 (#0D0D0F) — 地城深處的黑暗
- **次背景：** 暗紫灰 (#1A1625) — 帶有魔法氣息的暗色
- **主色調：** 火焰橙 (#FF6B35) — 代表 Boss 的威脅與緊迫感
- **輔助色：** 金色 (#FFD700) — 代表獎勵與重要資訊
- **成功色：** 翠綠 (#00D68F) — Boss 已被擊殺的狀態
- **文字色：** 淡灰白 (#E8E6E3) — 在暗色背景上清晰可讀

### Layout Paradigm
單頁式儀表板佈局，頂部為品牌標識與系統狀態，中間為 Boss 列表（卡片式），底部為指令輸入區。採用全寬設計，Boss 卡片以網格排列，重要的即將重生 Boss 置頂並放大顯示。

### Signature Elements
1. **發光邊框** — Boss 卡片根據重生倒數時間改變邊框發光顏色（紅→橙→綠）
2. **粒子背景** — 微妙的漂浮火星粒子動畫，營造地城氛圍
3. **脈動計時器** — 即將重生的 Boss 計時器帶有脈動動畫效果

### Interaction Philosophy
- 指令輸入模擬遊戲聊天框，帶有 ">" 提示符
- 操作回饋使用 toast 通知，樣式模擬遊戲系統訊息
- 卡片 hover 時微微浮起並增加發光強度
- 刪除操作需確認，模擬遊戲中的確認對話框

### Animation
- 頁面載入時 Boss 卡片依序淡入（stagger 50ms）
- 計時器數字變化時有微妙的 scale 彈跳
- 新增 Boss 時卡片從下方滑入
- 刪除時卡片向右滑出並淡出
- 背景粒子以極慢速度漂浮（CSS animation）

### Typography System
- **標題字體：** "Cinzel" — 中世紀風格的襯線字體，用於品牌名稱
- **數據字體：** "JetBrains Mono" — 等寬字體，用於計時器與指令
- **內文字體：** "Noto Sans TC" — 繁體中文正文，清晰易讀

### Brand Essence
**定位：** 天堂W公會專屬的 Boss 重生追蹤工具，為認真打王的玩家提供精準的時間管理。
**個性形容詞：** 精準、沉浸、專業

### Brand Voice
標題與 CTA 語氣如同資深公會幹部下達指令：簡潔、有力、帶有遊戲術語。
- 範例一：「掌握重生節奏，搶先一步到位」
- 範例二：「登記擊殺，倒數開始」

### Wordmark & Logo
一把燃燒的劍插在骷髏頭上的圖形標誌，象徵擊殺 Boss 的勝利。風格為扁平化但帶有微妙的光影效果。

### Signature Brand Color
**火焰橙 (#FF6B35)** — 這是整個品牌最具辨識度的顏色，代表 Boss 戰鬥的熱血與緊迫感。
